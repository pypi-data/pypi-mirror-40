Installing buzzword
==================

buzzword can be downloaded, installed and run locally.

Via Git
--------

```shell
git clone https://github.com/interrogator/buzzword
cd buzzword
python setup.py install
buzzword serve && buzzword open
```

Via pip
---------

```shell
pip install buzzword
buzzword serve && buzzword open
```
